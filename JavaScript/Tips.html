# 判断输入框是否为空

  • ES5：
if(input!==null||input!==undefined||input!=='')
1
  • ES6：

?? 运算符称为非空运算符,如果第一个参数不是假值（JS中的假值包括未定义 undefined、空对象null、数值0、空数字NaN、布尔false、空字符串）将返回第一个参数，否则返回第二个参数

if(input??''!=='')
